## Capturing Screenshots

1. Start the development environment: `docker compose up -d`
2. Open `http://localhost:3000` in your browser
3. Log in with demo credentials
4. Navigate to each page and capture screenshots (1920×1080 recommended)
5. Save as PNG files in this directory with the filenames listed above

For the CLI screenshot, use a terminal with a dark theme:

```bash
openpaper doctor
```

Capture the terminal output at 120×40 character size.
